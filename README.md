# Movie & TV Show App

A React-based **Movie & TV Show App** where users can display their favorite movies or TV shows, filter them by title or rating, and add new movies dynamically. The app uses **React Hooks** for state management and functional components.

## Features

- Display a list of movies/TV shows with title, description, poster, and rating.  
- Add new movies dynamically through a form.  
- Filter movies by **title** and **rating**.  
- Responsive UI with modern React functional components.  
- Uses **React Hooks** (`useState`, `useEffect`) for state management.  

---

## Tech Stack

- **React** – Frontend library for building the UI.  
- **React Hooks** – For state and effect management.  
- **CSS** – Styling the components.  
- **React Router DOM (optional)** – For page navigation.
